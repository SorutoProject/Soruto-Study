# Soruto Study
Soruto Studyは、学生向けに作られた、オープンソースのeラーニング(e-learning)環境です。  
e-ラーニングには以下の利点があります。

* 普段使っている、PCやスマホから簡単にアクセスできて、"勉強する"と決心したときにすぐに学習に取り組める。
* 自動採点なので、採点者が必要なく,また待ち時間がないため,勉強への集中力が切れない。

とまぁ、堅苦しいことを永遠と書いていたらキリがないので、実際に使ってみてください。
